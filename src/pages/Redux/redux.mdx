# Redux

Redux 被稱為狀態管理庫 (State Management Library)，目的在於管理和更新 Component 的狀態 (State)。

![redux](/redux-3.png)

在搭配 redux 使用 react 時會載入兩個不同的函式庫：

- redux：存放資料狀態的地方，當資料狀態改變的時候會有反應。
- react-redux：讓 redux 中的狀態可以和 React 相連接。
