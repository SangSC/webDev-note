# Object Methods

- 使用 methods 需要注意是否會修改原始資料。

## Object methods

1. 點號訪問：使用點號（`.`）來訪問物件的屬性，例如 `myObject.propertyName`。
2. 方括號訪問：使用方括號（`[]`）和屬性名來訪問物件的屬性，例如 `myObject['propertyName']`。
3. for...in 迴圈：使用 for...in 迴圈遍歷物件的屬性，並訪問每個屬性的值。
4. Object.keys ：使用 Object.keys 獲取物件的所有屬性名，並返回一個陣列。
5. Object.values ：使用 Object.values 獲取物件的所有屬性值，並返回一個陣列。
6. Object.entries ：使用 Object.entries 獲取物件的所有屬性及其對應的值，並返回一個二維陣列。
7. Object.getOwnPropertyNames ：使用 Object.getOwnPropertyNames 獲取物件的所有屬性名，並返回一個陣列，包括不可枚舉的屬性名。
