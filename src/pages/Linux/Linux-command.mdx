# Linux Command

## 文件操作

- **ls**：列出目錄內容。
- **pwd**：顯示當前工作目錄的路徑。
- **cd**：更改目錄。
- **touch**：創建空文件或更改文件的時間戳。
- **mkdir**：創建新目錄。
- **cp**：複製文件或目錄。
- **rm**：刪除文件或目錄。
- **rmdir**：刪除目錄。
- **ln**：創建文件或目錄的連接。
- **shred**：安全刪除文件。

## 文件查看和編輯

- **cat**：顯示文件內容。
- **nano**：以純文本模式編輯器打開文件。
- **vim**：強大的文本編輯器。
- **echo**：在標準輸出上顯示文本。

## 系統資訊

- **uname**：顯示系統信息。
- **neofetch**：顯示系統信息和藝術 ASCII 標誌。
- **cal**：顯示日曆。
- **free**：顯示記憶體使用情況。
- **df**：顯示文件系統使用情況。
- **ps**：顯示進程狀態。
- **top**：動態顯示進程信息。
- **htop**：交互式的進程查看器。
- **kill**：終止進程。
- **pkill**：根據名稱終止進程。
- **history**：顯示命令歷史。
- **reboot**：重新啟動系統。
- **shutdown**：關閉系統。

## 用戶管理

- **whoami**：顯示當前用戶名稱。
- **useradd**：創建新用戶。
- **adduser**：添加新用戶。
- **sudo**：以超級用戶權限運行命令。
- **su**：切換用戶。
- **exit**：退出登錄會話。
- **passwd**：修改用戶密碼。

## 系統管理

- **apt**：Debian 系統中用於管理軟件包的工具。
- **finger**：查看用戶信息。
- **man**：查看命令手冊。
- **whatis**：顯示命令的簡要描述。
- **curl**：通過 URL 進行數據傳輸。
- **zip**：壓縮文件。
- **unzip**：解壓縮文件。
- **less**：顯示文件內容，允許向前或向後瀏覽。
- **head**：顯示文件的開頭幾行。
- **tail**：顯示文件的末尾幾行。
- **cmp**：比較兩個文件的內容。
- **diff**：顯示兩個文件的差異。
- **sort**：對文件進行排序。
- **find**：在文件系統中搜索文件。
- **chmod**：更改文件的權限模式。
- **chown**：更改文件的所有者和組。
- **ifconfig**：顯示網絡接口信息。
- **ip address**：顯示 IP 地址信息。
- **grep**：搜索文本模式。
- **awk**：進階文本處理工具。
- **resolvectl status**：顯示系統 DNS 解析器的狀態。
- **ping**：測試與主機的連接。
- **netstat**：顯示網絡連接，路由表和接口統計信息。
- **ss**：顯示套接字統計信息。
- **iptables**：管理 IPv4 防火牆規則。
- **ufw**：簡化的防火牆配置工具。
- **systemctl**：系統服務管理器的控制命令。
